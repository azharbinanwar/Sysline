# Sysline

Free, open source macOS menu bar app for system visibility.

**v1 — Network module:** records per-app network usage over time so you can answer *"which app used my internet, and how much, over the last N days?"* — something macOS itself doesn't keep history for.

- Background polling via `nettop`
- Persistent history in SQLite
- Menu bar viewer: Today / Yesterday / 7 Days / 30 Days
- On-device only. No network calls, no analytics.

Records only while running. macOS 14+. Direct download, signed + notarized.
